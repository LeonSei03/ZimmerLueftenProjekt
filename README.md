# ZimmerLueftenProjekt
Small Projekt to regulate the time I open the window to let fresh air into my room. After my girlfriend always says I open the window for too long :) 

# ESP32 Lüftungs-Alarm
Projekt with:
- ESP32
- DHT Sensor
- Button
- LED or Buzzer

Funktion:
- save reference-temperature by pressing the button 
- trigger alarm if the temperature falls by too many degrees Celcius
